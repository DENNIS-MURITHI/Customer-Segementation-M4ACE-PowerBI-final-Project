# Customer-Segementation-M4ACE-PowerBI-final-Project
 A Power BI RFM analysis of 2,237 customers segmented into 6 groups — Champions to Lost. Key findings: top 36% of customers drive 78% of revenue, wines dominate at 50% of spend, and $89K in recoverable revenue identified. Built with DAX, Power Query, and an interactive 3-page dashboard.
